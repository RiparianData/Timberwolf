h1. Welcome to Timberwolf

Timberwolf is an open-source, command-line service to aid in importing emails from Microsoft Exchange into Hadoop hbase. It allows for periodic recynching between Exchange and hbase through periodic executions of the tool.

h2. Compiling Timberwolf

Compiling Timberwolf requires the following tools:

* Java JDK 1.6
* Apache Maven

To compile Timberwolf, run `mvn compile`.
To build a distribution, run `mvn package`.

The final Flume distribution artifacts will be in $project/target/.
